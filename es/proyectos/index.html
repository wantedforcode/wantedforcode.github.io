Lo que Hemos Entregado

Inicio Proyectos

Proyectos de Clientes Destacados

Proyectos de cliente, desde el briefing hasta producción. Ingenieros senior en cada uno. Acceso directo, responsabilidad total.

Hemos trabajado en SaaS, IoT, salud, gaming y agtech — dominios diferentes, mismos estándares de ingeniería.

SaaS / Integración

Plataforma Multi-Tenant de Contabilidad y ERP

SaaS empresarial que conecta sistemas de contabilidad con los principales ERPs (Sage, A3). Dashboards financieros en tiempo real, gestión documental, soporte multi-empresa e integración con LLM vía Model Context Protocol — construido sobre .NET 8, React y Azure.

.NET 8 · React · Azure · LLM/MCP

IoT / Monitoreo

Plataforma de Monitoreo IoT

Sistema de monitoreo en tiempo real para dispositivos IoT conectados. Métricas de uso, seguimiento de CO₂, alertas automáticas por email y generación de informes PDF — construido sobre .NET, Azure Functions y SQL Server.

.NET · Azure Functions · SQL Server

Healthcare / .NET Legacy

Auditoría y Modernización de SaaS Farmacéutico

Auditoría de seguridad para una plataforma SaaS farmacéutica multi-producto — farmacias, residencias y hospitales. Identificamos servidores EOL, datos reales de pacientes en entornos de prueba y exposición de APIs sin mapear — y entregamos una hoja de ruta de migración cloud con Docker y CI/CD.

Angular · .NET Framework · WCF · SQL Server

Otros Proyectos

Gaming Technology

Plataforma Marketplace para Jugadores

Plataforma startup que permite a los estudios de videojuegos ofrecer marketplaces integrados en el juego para contratar servicios especializados entre jugadores. Arquitectura multiplataforma con lógica compartida servidor/frontend, integración nativa en juegos y procesamiento de colas en tiempo real.

Nim · React · React Native · MongoDB · Redis

AgTech / Medioambiental

API de Cálculo de Huella Hídrica

API REST que calcula la huella hídrica e impacto ambiental de escenarios agrícolas. Procesamiento en segundo plano para cálculos de larga duración, integración meteorológica con Meteoblue, monitorización con Azure Application Insights y acceso a datos optimizado con EF Core.

.NET 8 · ASP.NET Web API · Azure · Entity Framework · Serilog
